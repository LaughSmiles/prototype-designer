## 2026年03月27日 10:17:13 - 双主题系统：Dark/Light 主题切换

### 目的
在暗色毛玻璃主题基础上，新增亮色主题，支持一键切换，并持久化用户偏好。

### 完成任务
- 将 CSS 变量系统从 `:root` 改为 `:root[data-theme="dark"]` / `:root[data-theme="light"]`
- 设计并实现 Light Glassmorphism 亮色主题配色方案
- 在工具栏添加主题切换按钮（太阳/月亮图标）
- 实现主题切换逻辑 + localStorage 持久化
- 将批注框、框选框等内联颜色改为 CSS 变量控制

### 关键决策
- **切换机制**: `data-theme` 属性 + CSS 变量，无需刷新页面
- **默认主题**: dark（与之前一致）
- **持久化**: localStorage key = `canvasEditor_theme`
- **亮色强调色**: 蓝色 #2563EB（更深沉专业）
- **亮色毛玻璃**: `rgba(255,255,255,0.75)` + `backdrop-filter: blur(20px)`
- **亮色画布**: 浅灰点阵网格 `rgba(0,0,0,0.08)`

### 修改文件
- `css/canvas-editor.css` - 双主题变量系统 + 批注颜色变量化
- `canvas-editor.html` - 工具栏添加主题切换按钮
- `js/canvas-editor.js` - 主题切换逻辑（initThemeToggle / applyTheme）
- `js/element-manager/element-annotation.js` - 移除内联颜色，改用 CSS class
- `js/canvas-view/canvas-view-box-selection.js` - 移除内联颜色，改用 CSS class

## 2026年03月27日 10:08:03 - UI重构：Dark Glassmorphism 暗色毛玻璃主题

### 目的
将画布编辑器的 UI 从浅色扁平风格升级为暗色毛玻璃专业风格，参考 Figma/Linear/Raycast 设计语言。

### 完成任务
- 全面重写 `css/canvas-editor.css`，引入 CSS 变量系统
- 实现暗色毛玻璃主题（Dark Glassmorphism）
- 更新 JS 内联样式适配暗色主题

### 关键决策
- **设计方向**: Dark Glassmorphism（暗色毛玻璃），参考 Figma/Linear/Raycast
- **背景层级**: 4层深色渐变（#0B0D11 → #0F1117 → #12151A → #181C23）
- **强调色**: 蓝色 #5B8DEF（替代旧 #3498db）
- **面板效果**: backdrop-filter: blur(20px) + 半透明背景
- **画布背景**: 网格点阵（radial-gradient）替代纯色
- **按钮颜色**: 语义化配色（绿=保存，红=删除，青=导出，黄=帮助）
- **圆角**: 统一使用 6/8/12/16px 四级圆角系统
- **动画**: 统一 100/200/300ms 三级时长

### 技术栈
- CSS 变量（Custom Properties）
- backdrop-filter 毛玻璃效果
- radial-gradient 网格背景
- CSS transition 动效

### 修改文件
- `css/canvas-editor.css` - 全面重写（~1200行 → ~1100行）
- `js/canvas-view/canvas-view-box-selection.js` - 框选颜色适配
- `js/element-manager/element-annotation.js` - 批注框颜色适配
